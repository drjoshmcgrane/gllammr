# Fit Statistics for Standard GLLAMM Models

Fit Statistics for Standard GLLAMM Models

## Usage

``` r
# S3 method for class 'gllamm'
fit(object, quiet = FALSE, ...)
```

## Arguments

- object:

  A gllamm object

- quiet:

  Suppress ICC computation messages (default: FALSE)

- ...:

  Additional arguments (not currently used)

## Value

Object of class `fit_gllamm` with fit statistics
