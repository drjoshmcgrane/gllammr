# Fit Statistics for Ordinal Regression Models

Fit Statistics for Ordinal Regression Models

## Usage

``` r
# S3 method for class 'gllamm_ordinal'
fit(object, test_po = TRUE, ...)
```

## Arguments

- object:

  A gllamm_ordinal object

- test_po:

  Test proportional odds assumption (default: TRUE for logit/probit)

- ...:

  Additional arguments

## Value

Object of class `fit_ordinal` with ordinal-specific fit statistics
